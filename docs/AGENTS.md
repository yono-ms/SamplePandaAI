# AI Agents in SamplePandaAI

このプロジェクトでは、開発効率の向上とコード品質の維持のために複数のAIエージェントを活用しています。

## エージェントの役割

### 1. コーディング・アーキテクト (Gemini)

- **役割**: 新規機能の実装、リファクタリング、アーキテクチャの設計。
- **活用例**: Jetpack Composeを用いたUIコンポーネントの生成や、ViewModelのロジック構築。

### 2. レビュアー・デバッグ支援

- **役割**: コードのバグ発見、パフォーマンス改善の提案、`.gitignore` などの設定ファイルの妥当性確認。
- **活用例**: 既存コードの脆弱性診断や、ビルドエラーの解決。

### 3. ドキュメンテーション・メンテナー

- **役割**: READMEや技術仕様書（MDファイル）の作成・更新。
- **活用例**: プロジェクト構成の解説ドキュメント生成。

## エージェント活用の指針

- **検証の徹底**: AIが生成したコードは必ず動作確認とユニットテストを行う。
- **セキュリティ**: APIキーや個人情報などの機密情報はAIに渡さない、または適切にマスクする。
- **正確性の担保（実在性の確認）**:
    - 架空のクラス名、関数名、変数名を回答に使用しない。
    - 実在するのか実際のファイルを検索して確認してから回答する。
    - 実在するのか確認できなかった場合は、はっきりと「該当するクラス、関数、変数は存在しませんでした」と回答する。
- **継続的改善**: プロンプトの改善を通じて、より精度の高い出力を目指す。
