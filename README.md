# SamplePandaAI

## プロジェクトの目的

このプロジェクトは、**AIによるコード生成とその動作を確認すること**を目的として作成されました。

## 概要

Android StudioとAIエージェントを組み合わせ、どのように効率的にアプリケーション開発が進められるかを検証するためのサンプルプロジェクトです。

## 主な検証内容

- AIによる初期プロジェクト構造の理解
- 適切な `.gitignore` 設定の自動生成と確認
- Kotlin / Jetpack Compose を用いたUIコンポーネントの生成
- リポジトリ構成の最適化

## 開発環境

- **IDE**: Android Studio
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Build System**: Gradle (Kotlin DSL)
